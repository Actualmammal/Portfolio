Overview

Reclamation Protocol is a still in development project where you create RTS units, by constructing them with factory game style construction lines, connecting buildings like miners, smelters and crafters via conveyor belts.

Contributions

I was the lead designer on Reclamation Protocol, so a lot of my time went into planning out how the game works. I designed all of the core systems including combat, resource processing, base building, and progression. Making sure they all fit together. This was actually very difficult and is something we are still working on.

Due to the short nature of each level it's a lot harder than normal to get the complexity level right, where each level is short enough to offer new challenges for the player, while still keeping the mechanics simple enough to manage both a factory and units at the same time. This is something we are still working on as we continue to improve and refine the game.

Other things I did on this project include:

  • Level design and creation, using tools our lead programmer created.
  • Discussing art direction with the artist to fit with gameplay.
  • Game balance of all units, production speeds, enemy difficulty etc.
  • Gameplay programming for self enclosed mechanics, as well as some help with programming the larger systems.

Prototype

I created the prototype for this game over a few days, and from that we decided to go forward with the project as a whole.
